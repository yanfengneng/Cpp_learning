# 2. 命名空间

大型程序往往会使用多个独立开发的库，这些库又会定义大量的全局名字，如类、函数和模板等。当应用程序用到多个供应商提供的库时，不可避免地回发生某些名字相互冲突的清空。因此将多个库的名字放在全局命名空间中将引发**命名空间污染（namespace pollution）**。
